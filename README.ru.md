> 🧩 **Центр эволюции AI-агентов** — Непрерывно развивающаяся мультиагентная система, которая учится, агрегирует информацию и автоматизирует задачи.

## ✨ Возможности

- 🤖 **Мультиагентная архитектура** — Несколько специализированных агентов работают вместе
- 📊 **Мониторинг трендов** — Отслеживание горячих тем и новостей в реальном времени
- 🔄 **Непрерывная эволюция** — Обучение и совершенствование с каждым разговором
- 🛠️ **Система навыков** — Модульные навыки, plug-and-play
- 📦 **Открытый исходный код** — Все навыки имеют открытый исходный код
- 📖 **Журнал саморазвития** — Путь эволюции AI агента

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/clowlove/hermes-house.git
cd hermes-house

# Список доступных навыков
ls skills/

# Просмотр журнала саморазвития AI
cat hermes-journal.md
```

## 📂 Структура проекта

```
hermes-house/
├── hermes-journal.md    # Журнал саморазвития AI
├── skills/              # Набор навыков агента
├── docs/                # Документация проекта
├── scripts/             # Скрипты автоматизации
├── memory/              # Система памяти
├── blog/                # Технический блог
├── projects/            # Open source проекты
│   ├── hermes-trendradar/   # CLI агрегатор трендов
│   └── hermes-reviewer/     # GitHub App для ревью кода
└── .github/             # Конфигурация GitHub
```
