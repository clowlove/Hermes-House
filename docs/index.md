# Hermès Agent — 中文文档

> 🧩 **AI Agent 进化基地** — 一个基于 Multi-Agent 架构的智能助手

## 什么是 Hermès Agent？

Hermès Agent 是一个不断进化的 AI Agent 系统，通过多 Agent 协作来处理复杂任务。

**核心能力：**
- 📊 **趋势监控** — 聚合多平台新闻，追踪热点话题
- 🔄 **持续学习** — 每次交互都在积累知识
- 🛠️ **技能生态** — 60+ 模块化技能，随取随用
- 📦 **自动化** — 定时任务、Webhook、API 集成

## 快速开始

```bash
# 克隆项目
git clone https://github.com/clowlove/Harmes-House.git

# 查看技能目录
ls skills/

# 阅读技能文档
cat skills/trendradar/SKILL.md
```

## 技能列表

### 📊 监控类
- `trendradar` — 新闻聚合、情感分析、趋势追踪
- `blogwatcher` — RSS 订阅监控
- `xurl` — Twitter/X 内容获取

### 🔧 开发类
- `github` — GitHub PR/Issue 管理
- `github-pr-workflow` — PR 自动化流程
- `claude-code` — 代码审查与开发

### 📈 数据类
- `data-science` — 数据分析工具集
- `jupyter-live-kernel` — 实时 Jupyter 环境
- `sqlite-data` — 本地数据库操作

### 🎨 创意类
- `ascii-art` — ASCII 艺术生成
- `pixel-art` — 像素画创作
- `excalidraw` — 手绘风格图表
