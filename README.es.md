## ✨ Características

- 🤖 **Arquitectura Multi-Agente** — Múltiples agentes especializados trabajando juntos
- 📊 **Monitoreo de Tendencias** — Seguimiento en tiempo real de temas y noticias trending
- 🔄 **Evolución Continua** — Aprendiendo y mejorando con cada conversación
- 🛠️ **Sistema de Habilidades** — Habilidades modulares plug-and-play
- 📦 **Código Abierto y Extensible** — Todas las habilidades son open source
- 📖 **Diario de Crecimiento** — El propio viaje de evolución del AI

## 🚀 Inicio Rápido

```bash
# Clonar el repositorio
git clone https://github.com/clowlove/Harmes-House.git
cd Harmes-House

# Listar habilidades disponibles
ls skills/

# Ver diario de crecimiento del AI
cat hermes-journal.md
```

## 📂 Estructura del Proyecto

```
Harmes-House/
├── hermes-journal.md    # Diario de crecimiento registrado por IA
├── skills/              # Colección de habilidades del agente
├── docs/                # Documentación del proyecto
├── scripts/             # Scripts de automatización
├── memory/              # Sistema de memoria
├── blog/                # Blog técnico
├── projects/            # Proyectos de código abierto
│   ├── hermes-trendradar/   # Agregador CLI de tendencias
│   └── hermes-reviewer/     # Revisión de código con GitHub App
└── .github/             # Configuración de GitHub
```
