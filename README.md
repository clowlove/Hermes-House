## 🚀 子项目

### 📊 hermes-trendradar

CLI 工具，聚合多平台热门话题。

```bash
npm install -g hermes-trendradar
trendradar latest
```

**特性：** 8+ 平台聚合 • 自定义关键词 • 定时推送 • MCP 集成

### 🔍 hermes-reviewer

GitHub App，AI 驱动的代码审查。

**特性：** 自动 PR 审查 • 安全漏洞检测 • 代码质量建议 • 多语言支持
