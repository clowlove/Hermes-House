# SEO Analysis

Full SEO audit: analyze keywords, meta tags, content quality, technical SEO, and competitor positioning.

## When This Skill Triggers

- "SEO audit", "analyze my SEO", "check rankings"
- "why is my traffic down", "traffic dropped"
- "keyword performance", "what keywords am I ranking for"
- "technical SEO", "indexing issues", "crawl errors"
- "improve my search visibility"

## What This Skill Does

1. **GSC Analysis** — Check Google Search Console for top queries, impressions, CTR
2. **Meta Tags Audit** — Title, description, Open Graph tags
3. **Content Analysis** — Headings, keyword density, readability
4. **Technical Check** — Robots.txt, sitemap, structured data, page speed
5. **Competitor Review** — Compare with top ranking competitors

## Quick Start

```
Audit SEO for https://shengtuo-tractor.com
```

```
Why is my traffic down? Check https://www.farm-implement.com
```

## Data Sources

- **Google Search Console** — Top queries, impressions, CTR by page
- **PageSpeed Insights API** — Core Web Vitals, performance scores
- **SERP Analysis** — Competitor title/meta patterns

## Key SEO Metrics

| Metric | Good | Warning | Poor |
|--------|------|---------|------|
| CTR | >3% | 1-3% | <1% |
| Position | <10 | 11-20 | >20 |
| LCP | <2.5s | 2.5-4s | >4s |
| CLS | <0.1 | 0.1-0.25 | >0.25 |

## Actionable Output

1. **Quick Wins** — 3-5 changes with biggest traffic impact
2. **30-Day Plan** — Prioritized action items
3. **Content Gaps** — Keywords competitors rank for that you don't
