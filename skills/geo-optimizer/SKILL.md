# GEO Optimizer

Optimize content for AI search engines (ChatGPT, Perplexity, Gemini, AI Overviews).

## When This Skill Triggers

- "rank in ChatGPT", "Perplexity optimization"
- "AI search", "GEO", "Generative Engine Optimization"
- "ChatGPT search", "AI Overviews"
- "appear in AI responses", "LLM optimization"
- "GEO strategy", "generative search"

## What This Skill Does

1. **Entity Recognition** — Identify entities AI models recognize
2. **Citation Analysis** — Find what sources AI cites for target queries
3. **Content Structuring** — Format for AI parsing and citation
4. **Schema Markup** — Add structured data for AI consumption
5. **E-E-A-T Signals** — Strengthen Experience, Expertise, Authority, Trust
6. **FAQ Optimization** — Target question-based queries AI answers

## GEO vs SEO Differences

| SEO Factor | GEO Focus |
|------------|-----------|
| Keywords | Entities & facts |
| Backlinks | Citations & references |
| Meta tags | Clear Q&A format |
| Content length | Concise, factual answers |
| Links | Source credibility |

## Key Tactics

1. **Cite authoritative sources** — AI prefers citing trusted content
2. **Structure as Q&A** — Clear question-answer format
3. **Use Schema markup** — FAQ, HowTo, Product schemas
4. **Add statistics** — Data-backed claims get cited more
5. **List sources** — "According to [source]" patterns
