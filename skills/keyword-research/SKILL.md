# Keyword Research

Discover, analyze, and prioritize keywords for SEO and GEO content strategies.

## When This Skill Triggers

- "find keywords", "keyword research", "keyword analysis"
- "search volume", "keyword difficulty"
- "content ideas", "what should I write about"
- "keyword opportunities", "topic clusters"
- "long-tail keywords", "question keywords"

## What This Skill Does

1. **Seed Keyword Expansion** — Generate related keywords from seed term
2. **Intent Classification** — Categorize by intent (informational, commercial, transactional)
3. **Difficulty Assessment** — Evaluate competition level
4. **Opportunity Scoring** — Prioritize by potential ROI
5. **Topic Clustering** — Group related keywords into clusters
6. **GEO Relevance** — Identify keywords likely in AI responses

## Quick Start

```
Research keywords for tractor exporters China
```

```
Find low-competition keywords for farm machinery
```

```
What keywords is [competitor] ranking for?
```

## Keyword Intent Types

| Intent | User Goal | Content Type |
|--------|-----------|--------------|
| Informational | Learn/Research | Blog posts, guides |
| Commercial | Compare/Evaluate | Comparisons, reviews |
| Transactional | Buy now | Product pages, landing pages |
| Navigational | Find a brand | Brand-specific pages |

## Data Sources

- Google Search Console (historical queries)
- SERP analysis (competition patterns)
- User-provided seed topics

## Actionable Output

1. **Keyword List** — Prioritized with volume, difficulty, intent
2. **Topic Clusters** — Grouped keywords for content hubs
3. **Content Calendar** — Dated publishing plan
