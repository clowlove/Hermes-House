# Meta Tags Optimizer

Optimize title tags, meta descriptions, Open Graph, and Twitter cards for maximum click-through rate.

## When This Skill Triggers

- "optimize title tag", "write meta description"
- "improve CTR", "click rate is low"
- "Open Graph tags", "social media preview"
- "fix my meta tags", "meta tags optimization"
- "title optimization", "SERP preview"

## What This Skill Does

1. **Title Tag Creation** — Compelling, keyword-optimized titles (50-60 chars)
2. **Meta Description Writing** — Click-worthy descriptions (150-160 chars)
3. **Open Graph Optimization** — Social sharing tags for Facebook/LinkedIn
4. **Twitter Card Setup** — Twitter-specific meta tags
5. **SERP Preview** — Show how it will appear in search results
6. **A/B Variations** — Multiple test variations

## Title Tag Best Practices

- Primary keyword first
- Brand name at end (if included)
- Unique per page
- No keyword stuffing
- Emotional/trigger words: "best", "top", "guide"

## Meta Description Best Practices

- Include primary keyword
- Clear value proposition
- Call to action
- Match search intent
- No duplicate descriptions
