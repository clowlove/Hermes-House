# Hermès TrendRadar

AI 驱动的热点话题聚合工具，支持多平台新闻追踪和情感分析。

## 安装

```bash
npm install hermes-trendradar
```

## 使用方法

```bash
# 查看热点话题
trendradar trending

# 聚合新闻
trendradar aggregate --query AI

# 情感分析
trendradar sentiment --topic AI
```

## 开发

```bash
npm install
npm run build
npm test
```
