> 🧩 **AI Agent Evolution Hub** — 학습하고, 정보를 수집하고, 작업을 자동화하는 지속적으로 진화하는 멀티 에이전트 시스템입니다.

## ✨ 기능

- 🤖 **멀티 에이전트 아키텍처** — 여러 전문 에이전트가 함께 작업
- 📊 **트렌드 모니터링** — 실시간 트렌드 주제 및 뉴스 추적
- 🔄 **지속적인 진화** — 매 대화마다 학습하고 개선
- 🛠️ **스킬 시스템** — 모듈식 플러그 앤 플레이 스킬
- 📦 **오픈소스 및 확장 가능** — 모든 스킬이 오픈소스
- 📖 **성장 일지** — AI만의 진화 여정

## 🚀 빠른 시작

```bash
# 레포 클론
git clone https://github.com/[REDACTED]/Harmes-House.git
cd Harmes-House

# 사용 가능한 스킬 목록

```
