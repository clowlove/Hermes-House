# Playwright

## 链接
https://playwright.dev

## 摘要
微软出品的端到端测试框架，支持 Python/JavaScript，可用于 Web 自动化、爬虫、视觉测试

## 用途/场景
- Web UI 自动化测试
- 反爬绕过
- 网页截图/视觉对比
- 浏览器自动化操作

## 评价
⭐⭐⭐⭐⭐

优点：
- 跨浏览器支持好（Chromium/Firefox/WebKit）
- API 设计优雅简洁
- 内置智能等待机制稳定
- 支持录制生成脚本

缺点：
- 资源占用较高
- 需要安装浏览器二进制

## 安装
```bash
pip install playwright
playwright install
```
